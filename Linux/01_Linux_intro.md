# Linux Introduction

## Linux kya hai?

Linux aik **open-source operating system** hai jo servers, cloud platforms, laptops, containers, networking devices aur bohat se modern systems mein use hota hai.

Simple lafzon mein:

**Linux woh system hai jo computer ko chalata hai aur user ko files, software, processes aur hardware manage karne deta hai.**

## Linux ka structure

Linux aik **hierarchical file system** use karta hai.

Matlab sab kuch aik tree structure mein hota hai:

```text
/
|-- home
|-- etc
|-- var
|-- usr
|-- tmp
```

Yahan `/` ko **root** kehte hain, aur ye poore system ka starting point hota hai.
